Wasteland Supply Terminal
Fallout 76 Resource Calculator

The Wasteland Supply Terminal is a tool that allows you to input as many recipes as you'd like, then automatically calculates every material you need for them. 
It also includes a recipe directory where you can see what recipes are available.
Please note that this Streamlit web version is TEMPORARY as I work on the official website.
